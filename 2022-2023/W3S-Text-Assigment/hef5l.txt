MySQL

MySQL SELECT - селектира дадена част от информацията в базата
MySQL WHERE - филтрира информацията по някакво условие
MySQL AND, OR, NOT - например where (условие) AND/OR (условие); например (дадено поле) IS NOT NULL
MySQL ORDER BY - подрежда информацията по някакъв критерии
MySQL INSERT INTO - вкарва информация в таблица
MySQL NULL Values - нещо не съществува (не е същото като просто нула)
MySQL UPDATE - променя нещо вече съществуващо в таблицата
MySQL DELETE - изтрива нещо съществуващо в таблицата
MySQL LIMIT - например LIMIT 1 ще ни изкара само 1 от селектираната информация, т.е. лимитира какво ни се извежда
MySQL MIN and MAX - минималната или максималната стойност в дадена колона
MySQL COUNT, AVG, SUM - брои по даден критерии, намира сума / средно аритметично на дадена колона
MySQL LIKE - сравнява и търси подобие/еднаквост между две стойности
MySQL Wildcards - аз знам 2 - "%" нула, един или няколко символа и "_" точно един символ
MySQL IN - някаква стойност да се съдържа в точно определен набор от стойности
MySQL BETWEEN - стойност да е между 2 граници
MySQL Aliases - ако таблицата например е с дълго име, се създава някакво съкращение, чрез което може да се обръщаме към нея
MySQL Joins - join-ва две таблици чрез някаква обща колона 
MySQL INNER JOIN - не съм го ползвала
MySQL LEFT JOIN - взима всичко от лявата таблица и добавя тези от дясната, които съвпадат
MySQL RIGHT JOIN - обратното left join 
MySQL CROSS JOIN - не съм го ползвала
MySQL Self Join - и това не съм ползвала
MySQL UNION - не съм го ползвала, но събира стойностите на няколко select-a
MySQL DISTINCT - ако имаме повтарящи се стойности, ще върне само тези, които са уникални
MySQL GROUP BY - групира стойности по някакъв критерий

MySQL Database - базата данни
MySQL Create DB - създава нова база данни, ако не съществува
MySQL Drop DB - drop-ва базата данни, ако съществува
MySQL Create Table - създаваме таблица, като подаваме име, ключ и стойностите, които ще са вътре - имената и типовете им
MySQL Drop Table - изтрива таблица
MySQL Alter Table - променя дадена част от таблица
MySQL Constraints - задава някакви граници за стойности в таблицата
MySQL Not Null - нещо да съществува
MySQL Unique - някаква стойност да може да се използва само веднъж
MySQL Primary Key - ключът, по който се добавят стойностите в таблицата и по който можем после да се обръщаме към тях, понеже е уникален
MySQL Foreign Key - ако дадена таблица наследява друга, по този начин свързваме ключа на първата, с този на втората
MySQL Check - проверка
MySQL Default - ако не се зададе специфична стойност, полето ще приеме тази стойност по подразбиране
MySQL Auto Increment - автоматично задава нова уникална стойност, когато добавяме в таблицата
