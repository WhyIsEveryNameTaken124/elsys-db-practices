MySQL SELECT - извежда данни от таблицата
MySQL WHERE - позволява поставяне на условие
MySQL AND, OR, NOT - логически операции И, ИЛИ и НЕ
MySQL ORDER BY - подрежда данните по дадена колона
MySQL INSERT INTO - добавяне на ред в таблицата
MySQL NULL Values - символизира празнота, не стойност 0 
MySQL UPDATE - променя даден запис
MySQL DELETE - изтрива даден запис
MySQL LIMIT - лимитира бройката на извежданите редове
MySQL MIN and MAX - връщат най-малката и най-голямата стойност в дадена колона
MySQL COUNT, AVG, SUM - преброяват, осредняват или сумират записите на дадена колона 
MySQL LIKE - сравнение на стрингове
MySQL Wildcards -  не знам
MySQL IN - проверя дали дадена стойност е в даден масив от данни
MySQL BETWEEN - дали дадена стойност е между дадени стойности
MySQL Aliases - задаване на алиас на дадена таблица или колона или друго
MySQL Joins - джойнва 2 таблици
MySQL INNER JOIN - джойнва записите на таблиците където съвпадат 
MySQL LEFT JOIN - джойнва записите на таблиците там където съвпадат към лявата таблица
MySQL RIGHT JOIN - джойнва записите на таблиците там където съвпадат към дясната таблица
MySQL CROSS JOIN - не знам
MySQL Self Join - не знам
MySQL UNION - не знам
MySQL DISTINCT - връща уникалните стойности в дадена колона
MySQL GROUP BY - групира по дадена колона

MySQL Database - датабаза
MySQL Create DB - създава датабаза
MySQL Drop DB - премахва датабаза
MySQL Create Table - създава таблица в дадена датабаза с дадени колони
MySQL Drop Table - премахва дадена таблица
MySQL Alter Table - променя дадена таблица
MySQL Constraints - ограничава стойностите на дадена колона
MySQL Not Null - забранява да е празно
MySQL Unique - няма как да има друг запис със същата стойност на тази колона; всеки запис има уникална стойност на тази колона 
MySQL Primary Key - главен ключ, id
MySQL Foreign Key - ключ към друга таблица
MySQL Check - условие, което да се спазва от данните в колоната
MySQL Default - стойност по подразбиране 
MySQL Auto Increment - автоматично увеличаване на стойността на всеки запис
