MySQL SELECT- избира таблицата, която трябва да покаже
MySQL WHERE-  това е условие, което филтрира по някакъв начин таблицата
MySQL AND, OR, NOT- това са бинарни операции и, или и не
MySQL ORDER BY- подрежда таблицата по някакъв специфичен начин
MySQL INSERT INTO- слага данни в таблицата, редове
MySQL NULL Values- това са данни, които отначало нямат никаква стойност
MySQL UPDATE- променя редове от таблицата 
MySQL DELETE- изтрива редове от таблицата
MySQL LIMIT- лимитира колко реда да се покажат от таблицата
MySQL MIN and MAX- взимат най-малката или голямата стойност от някоя колона в таблицата
MySQL COUNT, AVG, SUM- COUNT- брои, AVG- казва средно аритметичната стойност, SUM- сумира   всичко това в дадена колона
MySQL LIKE- проверява дали стойностите който му даваш са точно тези, които трябва, пр. точно 10 цифри или да има @ някъде
MySQL Wildcards- нещо лудо :)
MySQL IN- помага за where да станат нещата по-лесно (нещо такова беше)
MySQL BETWEEN-нз
MySQL Aliases- нз
MySQL Joins- обединява таблици
MySQL INNER JOIN- нз
MySQL LEFT JOIN- обединява се като първата таблица е отляво, а втората се залепя отдясно
MySQL RIGHT JOIN-  обединява се като първата таблица е отдясно, а втората се залепя отляво   (май)
MySQL CROSS JOIN- нз
MySQL Self Join- обединява се със себе си (ми нз)
MySQL UNION- залепя 2 резултата (каза го в час:)
MySQL DISTINCT- същото като unique
MySQL GROUP BY- групира слепените таблици по дадено нещо
 
MySQL Database- казваш коя database искаш да използваш
MySQL Create DB- правиш си база
MySQL Drop DB- дропва база (знам, че е тъпо, но не измислих нищо по-умно)
MySQL Create Table- създава се таблица
MySQL Drop Table- дропва таблица
MySQL Alter Table- дава да се променят колони
MySQL Constraints-  ми нз
MySQL Not Null- казва се че някой параметър от таблицата не трябва да е 0
MySQL Unique- казва се че някой параметър трябва да е уникален в цялата колона
MySQL Primary Key- това е ид то на таблицата 
MySQL Foreign Key- това трябва за да може да се обединят 2 таблици по общ ключ
MySQL Check- проверява за правилни данни още при инсъртването
MySQL Default- слага си дефолтно нещо
MySQL Auto Increment- автоматично си генерира
