MySQL SELECT - взима информацията от дадена таблица
MySQL WHERE - за проверка на данни
MySQL AND, OR, NOT - използват се в условие (AND - И, OR - ИЛИ, NOT - НЕ ТОВА)
MySQL ORDER BY - подрежда в възходящ или низходящ ред
MySQL INSERT INTO - вкарваме стойности в таблицата
MySQL NULL Values - представляват невъведено поле в таблицата
MySQL UPDATE - ъпдейтваме таблицата с нови стойности
MySQL DELETE - изтриване на ред от таблицата
MySQL LIMIT - лимитира до определен брой стойности
MySQL MIN and MAX - най-малка и най-голяма стойност
MySQL COUNT, AVG, SUM - преброява стойностите за един ред, намира средната стойност, сумира стойностите
MySQL LIKE - проверява дали са еднакви две стойности
MySQL Wildcards - идея си нямам
MySQL IN - проверява дали в дадена стойност е част от стойностите, които са му дадени
MySQL BETWEEN - идея си нямам
MySQL Aliases - идея си нямам
MySQL Joins - съединява всички елементи от две таблици
MySQL INNER JOIN - съединява две таблици като взима само свързаните елементи от двете таблици
MySQL LEFT JOIN - съединява две таблици като взима всички елементи от първата таблица и залепя за тях елементите свързани с първата таблица от втората таблица
MySQL RIGHT JOIN - съединява две таблици като взима всички елементи от втората таблица и залепя за тях елементите свързани с втората таблица от първата таблица
MySQL CROSS JOIN - идея си нямам
MySQL Self Join - идея си нямам
MySQL UNION - залепва два резултата от query
MySQL DISTINCT - връща само по една от няколко еднакви стойности
MySQL GROUP BY - групира стойности при определено условие

MySQL Database - съхранява всички таблици и информация
MySQL Create DB - създава датабазата ако не съществува
MySQL Drop DB - изтрива датабазата
MySQL Create Table - създава таблица
MySQL Drop Table - изтрива таблица
MySQL Alter Table - добавяне на редове при вече създадена таблица
MySQL Constraints - идея си нямам
MySQL Not Null - стойността да не е нулева
MySQL Unique - стойността да е уникална
MySQL Primary Key - Главният ключ в таблицата (id)
MySQL Foreign Key - Ключ с който се свързва с главният ключ на друга таблица
MySQL Check - проверяваме стойостите при дадено условие
MySQL Default - предполагам че задава дефоутна стойност
MySQL Auto Increment - автоматично да се увеличава
